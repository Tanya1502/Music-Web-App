Music-Web-App

Tech Stack

Client: React JS
Server: Node JS, Express JS
Database: Mongo DB

Run Locally

Clone the project
git clone 

Go to the project directory
cd Music-Web-App

Install dependencies
  cd backend
  npm install
  cd frontend
  npm install

Start the server
  cd backend
  node index.js
  
Start the Client
  cd frontend
  npm start
